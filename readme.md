# jGDSync
* My Personal Google Drive Synchonisation tool
* Status: in Developing


# How to use
* Build the app
  * gradlew.bat build, after this command you will find in the target folder a JAR File
  * gradle
* Currently you must provide your own client
  * Go to step 1 on page https://developers.google.com/drive/api/v3/quickstart/java and follow the instructions

# Credentials
* the app stores your credentials in [USER_HOME]/.credentials/jGDSync/StoredCredential
